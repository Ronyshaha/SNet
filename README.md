# YOLO-SNet: Detection of Distant Smaller Entities from Aerial Surveillance Images of Heterogeneous Objects
## Overview
This repository contains the implementation of the manuscript:
**YOLO-SNet: Detection of distant smaller entities from aerial surveillance images of heterogeneous objects**
## This paper Authors
- Rony Shaha
- Kaushik Sarker* 
- Md Mahibul Hasan
- Sayed Jobaer
- Foysal Ahmed
- MD Ahasan Habib Tushar
- Sumonta Ghosh
## Paper Abstract
Small object detection in aerial surveillance imagery is challenging due to low resolution, scale variation, cluttered backgrounds, and heterogeneous object appearances. In this work, we propose **YOLO-SNet**, a framework designed to improve the detection of distant small objects from UAV imagery.
